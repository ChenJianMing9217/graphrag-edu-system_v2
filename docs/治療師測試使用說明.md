# EduBot 早療諮詢系統 — 治療師測試使用說明

感謝您撥空協助測試本系統！您的回饋對於系統優化與後續研究極為重要。

---

## 一、系統定位

**EduBot** 是一個基於兒童早療聯合評估報告的 AI 諮詢系統。透過對話形式，協助家長 / 治療師：

- 解讀評估報告（分數、能力剖面）
- 取得居家訓練建議
- 查詢轉介資源與補助
- 獲得情緒支持與家庭溝通建議

---

## 二、登入與開始使用

1. 開啟瀏覽器，前往：`http://[系統位址]`
2. 使用您的測試帳號代碼登入
3. 系統會自動載入該帳號對應的兒童評估報告
4. 在輸入框輸入問題，按送出即可

---

## 三、可以問哪些類型的問題？

我們設計系統涵蓋 14 種早療常見諮詢情境（系統會自動辨識）：

| 類型 | 範例 query |
|------|-----------|
| **報告總覽** | 「這份報告的重點是什麼？」「我該先看哪一段？」 |
| **分數解讀** | 「百分位 16% 代表什麼？」「標準分 85 算嚴重嗎？」 |
| **臨床觀察** | 「核心肌力不足平常會怎麼表現？」「協調不佳是什麼意思？」 |
| **能力剖面** | 「孩子的優先項目是什麼？」「先練哪兩個比較好？」 |
| **居家訓練** | 「在家可以怎麼練？」「一天大概多久？」 |
| **融入日常** | 「上學趕怎麼塞訓練時間？」「玩耍時可以練什麼？」 |
| **早療評估需求** | 「需要做物理治療嗎？」「一週幾次？」 |
| **轉介資源** | 「附近哪裡有早療？」「物理治療要去哪掛？」 |
| **隱私分享** | 「報告可以給老師嗎？」「要遮哪些？」 |
| **學校合作** | 「怎麼跟老師說？」「體育課可以怎麼調整？」 |
| **補助申請** | 「早療補助怎麼申請？」「需要哪些文件？」 |
| **後續追蹤** | 「多久追蹤一次？」「怎麼判斷有進步？」 |
| **情緒支持** | 「家人不支持治療怎麼溝通？」「我很焦慮怎麼辦？」 |
| **進步查詢** | 「跟上次比有進步嗎？」「分數變好的部分？」 |

## 四、最重要：請給每個回答打分

**每個 EduBot 回應下方都有兩個按鈕**：

```
👍 有幫助    👎 無幫助
```

請依您的專業判斷點選：
- **👍 有幫助**：回答內容正確、實用、有建設性
- **👎 無幫助**：答非所問、錯誤資訊、過於空泛、不夠專業

> 您的反饋會被記錄並用於系統優化。**這是我們最珍貴的研究資料**，請多花一秒鐘點選。

## 五、系統的限制（請有心理準備）

### 5-1 不在系統範疇內的話題會被引導

如果您問了與兒童早療無關的問題（例如：天氣、投資、別人的事），系統會：
1. 簡短回應您的問題
2. 說明本系統專注於早療諮詢
3. 邀請您回到早療相關話題

這是設計如此，請放心 —— 我們**不希望系統不懂裝懂**。

### 5-2 缺地區資訊時會主動詢問

如果您問轉介資源或補助但**沒有提到縣市**，系統會：
1. 先給全台通用資訊
2. 詢問您所在縣市以提供更具體建議

### 5-3 多種可能的問題會給綜合答案

如果您的 query 同時涉及多個面向（如「孩子這樣怎麼辦」），系統會：
1. 給綜合性的多方向回答
2. 詢問您想深入了解哪個方向

### 5-4 已知不夠完美的部分

- 連續 5 輪以上的長對話可能會有跨輪推理失誤
- 罕見的醫學術語可能識別不準
- 情緒性問題的回應風格還在優化

**遇到任何上述問題，請務必按 👎 並（如果方便）寫下您的觀察**。

## 六、測試建議流程

每天測試 15-30 分鐘即可，請涵蓋以下情境：

### 流程 A：報告解讀流（5-10 分鐘）
1. 「這份報告的重點是什麼？」
2. 「百分位 X 是什麼意思？」
3. 「優先要關注哪些項目？」

### 流程 B：訓練建議流（5-10 分鐘）
1. 「在家可以怎麼練 [某能力]？」
2. 「一天多久比較合適？」
3. 「上學前怎麼塞練習時間？」

### 流程 C：轉介流（5-10 分鐘）
1. 「我住在 [縣市]，附近哪裡有早療？」
2. 「需要先看哪一科？」
3. 「補助怎麼申請？」

### 流程 D：跨主題流（10-15 分鐘）
測試系統能否流暢處理話題切換：
1. 從「分數解讀」切到「居家訓練」
2. 從「轉介資源」切到「補助申請」
3. 從「情緒支持」切到「家庭溝通」

### 流程 E：邊界測試（5 分鐘）
故意試試：
1. 問非常簡短的問題：「然後呢？」「為什麼？」
2. 問模稜兩可的問題：「孩子這樣怎麼辦？」
3. 問與早療無關的問題：看系統如何引導
