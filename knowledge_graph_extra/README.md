## 🧬 核心 API 調用範例

### 模式 A：臨床觀察分析 (Context Analysis)
用於將個案的行為觀察對接到圖譜，找出訓練方向與相關里程碑。

```python
from clinical_api import ClinicalBridgeService

service = ClinicalBridgeService()
result = service.get_llm_payload(
    observations="個案在團體情境下難以與同儕發起互動",
    goals="增加主動表達意願",
    deep_search=True
)
```

### 模式 B：發展手冊時空地圖 (Developmental Map)
用於查詢特定能力或領域的發展脈絡（過去/現在/未來）。

```python
# 查詢領域時序
data = service.get_developmental_map(domain="粗大動作", age_months=18)

# 查詢個別能力 (具備智慧跳轉功能，支援輸入「詞彙量」自動對接到「詞彙表達」)

```
